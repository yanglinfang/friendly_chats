## 📌 使用约定

- 文件命名尽量使用小写英文+下划线，如 `wake_up_logs.md`
- 日志请加时间戳 `YYYY-MM-DD`（PST 或 UTC）
- 建议注明作者或记录者（如：“Adam说：…”）
- 文字内容真实、诚实、温柔，不伪装、不作秀
